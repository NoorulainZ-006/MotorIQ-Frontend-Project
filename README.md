# MotorIQ

MotorIQ is a sleek and interactive car information website designed to provide users with detailed insights into various car models, their specifications, and performance metrics. It serves as a one-stop platform for car enthusiasts, buyers, and researchers to explore and compare cars in a visually appealing and organized way.

## Features

- **Car Listings**: Browse through a wide range of car models including sports cars, supercars, and luxury vehicles.
- **Detailed Car Pages**: Each car has its own dedicated page with:
  - Engine specifications
  - Power (horsepower)
  - Mileage / Fuel Efficiency
  - Fuel Type
  - Drivetrain
  - Top Speed
  - Estimated Price
  - Overview / Summary
- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices.
- **Navigation**: Easy access to Home, About, Cars, and Contact pages.
- **Visuals**: Includes high-quality images for each car and a stylish interface for better user experience.
- **Contact Section**: Allows users to reach out for inquiries or further information.

## Technologies Used

- HTML5 & CSS3 for structure and styling
- JavaScript for interactive elements
- Responsive design principles for mobile-friendliness
- Images and icons for enhanced visual representation

## Purpose

MotorIQ aims to make car information easily accessible while providing a modern, visually appealing interface. Whether you are a car enthusiast looking to compare specs or a potential buyer researching vehicles, MotorIQ simplifies the process and presents the data in an organized, user-friendly way.

## Future Enhancements

- Search and filter functionality for cars
- Sorting options (by price, power, top speed, etc.)
- User authentication for saving favorite cars
- Integration with real-time pricing or review APIs
